# 핵심 요약
- LLM의 환각(Hallucination)을 일회성 오류가 아닌, 시간에 따라 진화하는 '잠재 상태(Latent State)'로 재정의함.
- 장기 추론(CoT) 과정에서 환각이 발생하는 시점과 추론 궤도 이탈 과정을 추적하는 'Streaming' 방식의 탐지 제안.
- 국소적 오류(Step-level)와 전역적 상태(Prefix-level)를 구분하여, 환각의 누적과 논리적 붕괴 과정을 구조적으로 설명함.
- 단순 탐지를 넘어 실시간 개입, 추론 중단, 재생성 제어와 같은 능동적 대응의 이론적 토대를 마련함.

# 상세 내용
- 기존 환각 탐지의 한계
  - 주로 최종 답변이나 개별 단계의 사실 관계에만 집중함.
  - 장기 CoT에서는 각 단계가 개별적으로는 그럴듯해 보여도 전체 맥락이 점진적으로 왜곡되는 현상을 포착하기 어려움.

- 환각의 새로운 정의: 잠재 상태(Latent State)
  - 환각을 "어디서 틀렸는가"라는 이벤트 중심이 아니라, "언제부터 신뢰 가능한 궤도를 이탈했는가"라는 과정 중심으로 파악함.
  - 일시적 오류 후의 회복 가능성이나, 사소한 오류가 누적되어 발생하는 시스템적 붕괴를 설명할 수 있음.

- 이중 신호 체계
  - 단계 수준(Step-level): 각 추론 단계가 새로운 오류나 근거 없는 정보를 도입했는지 판단하는 국소적 신호.
  - 접두 수준(Prefix-level): 현재까지의 전체 추론 경로가 환각에 의해 지배되고 있는지를 나타내는 전역적 상태.

- 기술적 방법론
  - 시간 인지적 지수 가중치: 긴 시퀀스에서 후반부 신호가 희석되는 문제를 해결하기 위해 현재 단계 토큰에 민감하게 반응하도록 설계함.
  - 논리적 제약 기반 추정: 환각 상태로의 진입은 민감하게 포착하되, 다시 신뢰 상태로 회복하기 위해서는 반복적이고 안정적인 정정 증거를 요구함.

- 연구의 의의 및 결과
  - AUC와 같은 정적 지표뿐 아니라 환각 발생 지연, 회복 속도 등 추론 동역학 지표에서 우수한 성능을 보임.
  - 신뢰 가능한 언어 모델을 위한 실시간 모니터링 및 제어 가능성을 제시함.

# 인사이트 / 할 일
- 인사이트: LLM의 답변을 검증할 때 단답형의 진위 파악보다, 추론의 '흐름'이 깨지는 임계점을 찾는 것이 더 고도화된 접근법임을 깨달음. 
- 인사이트: "신뢰를 잃기는 쉽지만 다시 얻기는 어렵다"는 인간의 심리적 기제를 모델의 환각 회복 알고리즘(엄격한 정정 증거 요구)에 반영한 점이 인상적임.
- 할 일: 긴 추론이 필요한 에이전트 설계 시, 중간 단계마다 '신뢰 궤도'를 이탈했는지 체크하는 가드레일 로직 검토하기.
- 할 일: 해당 논문(Streaming Hallucination Detection in CoT)의 원문을 찾아 구체적인 수식과 구현 방법 확인하기.
