### CSS를 모듈처럼 사용하기

기존엔 HTML 파일에서 `<link>` 태그로 CSS를 불러왔음. 웹팩을 사용하면 JS에서 CSS를 import할 수 있음. 이렇게 하면 CSS도 모듈처럼 관리할 수 있고, 필요한 스타일만 번들에 포함시킬 수 있음.

```js
// CSS를 import하면 웹팩이 알아서 처리해요
import "./style.css";
```

---
### 1. CSS 로더 설치하기

웹팩이 CSS를 처리할 수 있도록 필요한 로더들을 설치.

```bash
npm install --save-dev style-loader css-loader
```

- `css-loader`: CSS 파일을 JS에서 import 할 수 있는 형태로 바꿈.
- `style-loader`: 변환된 CSS를 브라우저 실행 시 `<style>` 태그로 만들어 동적으로 삽입함.

---
### 2. 웹팩에 CSS로더 설정 추가하기

`webpack.config.js` 파일에 CSS를 처리하는 로더를 추가.

```js
module.exports = {
  // ... 기존 설정 유지
  module: {
    rules: [
      // ... 기존 rules 유지
      {
        test: /\.css$/, // .css 파일을 처리해요
        use: [
          'style-loader', // CSS를 <style> 태그로 주입해요
          'css-loader' // CSS를 JavaScript 모듈로 변환해요
        ]
      }
    ]
  }
};
```
