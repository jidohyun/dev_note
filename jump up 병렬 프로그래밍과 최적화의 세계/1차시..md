### 목차

- 행렬의 컴퓨터 표현 이해
- 합성곱 연산자 이해
- 실습환경 프레임워크 구축
- 직접 구현해보는 합성곱
- 다양한 커널 구현

### 프로그램 최적화?

- 프로그램이 자원을 효율적으로 사용하게 만드는 과정

### 최적화의 기본

- 최적화는 크게 두가지의 방법론이 존재
	- 알고리즘을 개선하는 방법
	- 구현을 개선하는 방법

### 알고리즘

- 알고리즘이란 입력과 출력이 주어졌을 때 그것을 수행하는 방법
- 동일한 입력과 출력을 갖는 여러 알고리즘이 존재할 수 있음.

### 구현 개선

- 구현이란 특정 알고리즘의 실제 동작 가능한 코드
- 같은 알고리즘이어도 여러 구현 방법이 있을 수 있음
